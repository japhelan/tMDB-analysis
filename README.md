# tMDB-analysis

Respository for a project I did involving EDA on the tMDB-5000 dataset

data contains the database files + CPI index from the past ~75 years
notebooks currently contains the notebook I did all of the EDA in
output contains the adjusted dataset after analysis was completed

An accompanying EDA presentation can be found [here](https://github.com/japhelan/tMDB-analysis/blob/main/EDA%20Presentation.pdf)
